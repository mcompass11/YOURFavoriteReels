Objective

This application will serve to give users access to a variety of information on movies. Not just that but users will have the opportunity to sign up and create a list of their favorites and update their personal information.

URL Endpoints

Request: Get a list of all movies URL: /movies Repsonse: A JSON holding data of all movies

Request: Get data about movie by title URL: /movies/name Repsonse: A JSON holding data on specific movie

Request: Get data on specific genre URL: /genres Repsonse: A text message of description of genre

Request: Get data on director by name URL: /directors Repsonse: A text message of description of director

Request: Allow users to register URL: /users Repsonse: A JSON holding data on added user

Request: Allow users to update their info URL: /users/profile Repsonse: A text message indicating user info updated

Request: Allow users to add movie to their list URL: /users/movies Repsonse: A JSON holding data of movies with newly added

Request: Allow users to remove movie from their list URL: /users/movies Repsonse: A JSON holding data of movies without newly removed

Request: Allow user to deregister URL: /users Repsonse: A text message indicating user removed
