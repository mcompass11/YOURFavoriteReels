Objective

This application will serve to give users access to a variety of information on movies. Not just that but users will have the opportunity to sign up and create a list of their favorites and update their personal information.

URL Endpoints

Request: GET, Opens the default page URL: / Repsonse: A text message stating welcome

Request: GET, Get a list of all movies URL: /movies Repsonse: A JSON holding data of all movies

Request: GET, Get data about a movie by title URL: /movies/:Title Repsonse: A JSON holding data on specific movie

Request: GET, Get data on specific genre URL: /genre/:Name Repsonse: A JSON holding data on a specific genre

Request: GET, Get data on director by name URL: /director/:Name Repsonse: A JSON with the description of director

Request: GET, Returns a list of users URL: /users Repsonse: A JSON holding data on all users

Request: POST, Allows users to register URL: /users Repsonse: Returns JSON of info on the user added, notifies if username is already added

Request: PUT, Allows user to update their info URL: /users/:Username Response: A JSON with the included changes

Request: PUT, Allow users to add movie to their list URL: /users/:Username/Movies/:MovieID Repsonse: A JSON holding data of user info with newly added movie

Request: DELETE, Allow users to remove movie from their list URL: /users/:Username/Movies/:MovieID Repsonse: A JSON holding data of user info with movie removed

Request: DELETE, Allow user to deregister URL: /users/:Username Repsonse: A text message indicating user removed
